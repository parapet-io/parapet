# parapet-pario

`parapet-pario` is a small executor-backed reference runtime for Parapet.

It contains:

- `ParIO`
- `ParIORuntime`
- `ParIORuntimeConfig`
- `ParIOApp`
- ParIO runtime docs and workbook
- ParIO-specific runtime tests
- conformance specs from `parapet-testkit` run against `ParIO`

ParIO is intended for examples, tests, learning, and runtime experimentation. It is not intended to replace production
effect runtimes such as Cats Effect, ZIO, or Reactor.
